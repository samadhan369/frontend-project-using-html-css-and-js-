CSS Questions Answers

Explore common CSS questions and answers to enhance your styling skills.



1. What is CSS used for?

CSS (Cascading Style Sheets) is used to style and layout web pages, including adjusting colors, fonts, spacing, and more.


2. What is the difference between inline, internal, and external CSS?

Inline CSS is applied directly to an element via the style attribute. Internal CSS is written inside a <style> tag within the <head>. External CSS is stored in a separate file linked to the HTML document.


3. What is a CSS selector?

A CSS selector is a pattern used to select and style elements in HTML. For example, .class, #id, and element are common selectors.


4. What is the difference between relative, absolute, and fixed positioning in CSS?

Relative positioning adjusts an element's position relative to its normal flow. Absolute positioning places an element relative to its nearest positioned ancestor. Fixed positioning positions an element relative to the browser window.


5. What are pseudo-classes in CSS?

Pseudo-classes define the special state of an element, such as :hover, :focus, and :nth-child().


6. What is the difference between em and rem units in CSS?

The em unit is relative to the font size of its nearest parent, while rem is relative to the root element's font size.


7. How do you create a CSS grid layout?

Use the display: grid; property on a container element and define rows and columns using grid-template-rows and grid-template-columns.


8. What is a media query in CSS?

A media query allows you to apply styles based on the device's screen size or other characteristics, such as @media (max-width: 768px).


9. What is the z-index property in CSS?

The z-index property controls the stack order of elements. Higher values are displayed in front of lower values.


10. What is the difference between flexbox and CSS grid?

Flexbox is designed for one-dimensional layouts (rows or columns), while CSS Grid is for two-dimensional layouts (rows and columns).


11. What is the difference between inline, block, and inline-block elements in CSS?

Inline elements do not start on a new line and take only the necessary width. Block elements start on a new line and take up the full width available. Inline-block elements behave like inline elements but allow block-level styling.


12. What is the difference between relative and absolute units in CSS?

Relative units (e.g., em, rem, %) are based on another value. Absolute units (e.g., px, cm, in) have a fixed size regardless of context.


13. How do you add a custom font in CSS?

Use the @font-face rule to define a custom font and specify its source file. Then use the font-family property to apply it.


14. What is the difference between padding and margin?

Padding is the space between the content and its border, while margin is the space outside the border between the element and other elements.


15. What is the purpose of the overflow property in CSS?

The overflow property controls how content is handled when it overflows the dimensions of its container. Options include visible, hidden, scroll, and auto.


16. How do you create a transition in CSS?

Use the transition property to define the timing, duration, and effect for changes in CSS properties.


17. What is the difference between position: sticky and position: fixed?

position: sticky allows an element to act as a relative element until it crosses a threshold, after which it behaves like fixed. position: fixed keeps an element in place relative to the viewport.


18. How do you center a div in CSS?

You can use Flexbox with display: flex and justify-content: center, align-items: center, or set margin: auto on a block element with a specified width.


19. What is the box model in CSS?

The CSS box model describes the rectangular boxes generated for elements. It includes the content, padding, border, and margin of an element.


20. How do you make a website responsive using CSS?

Use media queries, flexible grid layouts, relative units like % or em, and ensure images and other media scale properly using max-width: 100%.


21. What is the purpose of the z-index property in CSS?

The z-index property determines the stack order of elements. Higher values are displayed in front of lower values.


22. How do you apply styles to specific child elements in CSS?

You can use child combinators (>) or nth-child pseudo-classes like :nth-child() or :nth-of-type().


23. What is the difference between absolute and relative positioning in CSS?

absolute positioning places an element relative to its nearest positioned ancestor. relative positioning moves an element relative to its normal position.


24. What are pseudo-classes in CSS?

Pseudo-classes define the special state of an element, such as :hover for mouse hover or :nth-child() for specific children.


25. What is the difference between max-width and min-width in CSS?

max-width sets the maximum width an element can occupy, while min-width sets the minimum width an element can occupy.


26. What is the difference between inline and inline-block elements?

Inline elements cannot have width or height specified, while inline-block elements retain their inline flow but allow width and height to be set.


27. How do you make text bold in CSS?

Use the font-weight property with the value bold or a numeric weight such as 700.


28. What is the difference between visibility: hidden and display: none in CSS?

visibility: hidden hides the element but keeps its space. display: none removes the element from the layout flow.

30. What is the purpose of the clip-path property in CSS?
