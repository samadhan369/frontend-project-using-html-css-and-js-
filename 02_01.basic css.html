Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style and format the layout of web pages. With CSS, you can change the color, fonts, layout, and other aspects of the page's design. CSS is a fundamental part of web development, and mastering it allows for the creation of beautiful and responsive websites.

Some key concepts to understand in CSS include:

  • Selectors: Identify HTML elements to style
  • Properties: Define the style applied to elements
  • Values: Specify the values for the properties

CSS Selectors

Selectors are used to target specific HTML elements that you want to style. Here are some common types of selectors:

  • Element Selector: Selects all elements of a specific type. Example: p { color: red; }
  • Class Selector: Selects elements with a specific class. Example: .myClass { font-size: 16px; }
  • Id Selector: Selects an element with a specific id. Example: #myId { background-color: blue; }
  • Universal Selector: Selects all elements on the page. Example: * { margin: 0; }

CSS Properties

CSS properties define how an element is styled. Some common properties include:

  • Color: Used to set the color of text. Example: color: blue;
  • Background: Sets the background color or image. Example: background-color: #f4f4f4;
  • Font-size: Controls the size of the text. Example: font-size: 14px;
  • Padding: Sets the space inside the element. Example: padding: 10px;
  • Margin: Sets the space outside the element. Example: margin: 20px;

The CSS Box Model

The Box Model is a fundamental concept in CSS, as it defines the space around an element. Every element on a webpage is a rectangular box, and CSS properties are used to control the size and spacing of that box.

  • Content: The actual content of the element.
  • Padding: The space between the content and the border.
  • Border: The boundary around the padding (if any).
  • Margin: The space outside the border.
