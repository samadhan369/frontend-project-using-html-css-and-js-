HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses a system of tags to structure and format content on the web.

HTML Structure

An HTML document is made up of elements that define the structure of a webpage. Below is an example of a basic HTML structure:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to HTML!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>

Key HTML Elements

  • <h1> to <h6>: Headings, with <h1> being the largest and <h6> the smallest.
  • <p>: Paragraphs for text content.
  • <a>: Links to navigate to other pages or websites.
  • <img>: To display images.
  • <ul> and <ol>: Unordered and ordered lists.
  • <table>: To create tables with rows and columns.
  • <form>: To collect user input via text boxes, buttons, and other controls.

HTML Attributes

Attributes provide additional information about elements. Examples include:

  • href: Specifies the link in an <a> tag.
  • src: Specifies the source file for an image or script.
  • alt: Provides alternate text for an image.
  • id: Assigns a unique identifier to an element.
  • class: Assigns one or more class names for styling or scripting.
